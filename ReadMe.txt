一：运行工具
Pycharm
二：运行环境
项目自带程序运行需要的所有的包（是一个虚拟环境），使用Pycharm打开即可运行。
三：运行方式
直接运行Predict.py文件即可
四：数据
Data文件夹下面是我用于测试的三个航班，是2019-1-1至2019-12-31的所有航班的价格及对应的观测日期。
五：结果
得到的结果可以通过访问:127.0.0.1:5000得到测试数据，方便前端通过接口获得值
比如：浏览器输入127.0.0.1:5000/?dep_city=PEK&arrive_city=HGH便可以返回2020-1-1至2020-12-31日，从北京到广州所有航班的最低价格，及观测日期还有往前20天，往后40天的机票价格数据。
